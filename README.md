A collection of custom skills for Claude Code.

## Skills

**Built by me:**
- `blender` – Blender best practices and Python snippets for the Blender MCP server
- `recording-app-demo-videos` – Automated app demo video recording with Playwright

**Sourced from the community:**
- `code-audit` – Codebase research and code auditing
- `refactor-clean` – Code refactoring using SOLID principles and clean code patterns
- `tailwindcss` – Tailwind CSS v4 patterns for Next.js applications

If you authored one of the community-sourced skills and would like attribution, please open an issue and I will happily fix this (unfortunately I can't remember the sources for the non-attributed skills on this list).
